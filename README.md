## Description

[Nest](https://github.com/nestjs/nest) boilerplate

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```

## Usage

Make a config folder to use the config and create a yaml file in it.

```
config
├── default.yaml
├── development.yaml
└── production.yaml
```
